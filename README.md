## How to use
* Download and Install Python 
* Install the required libraries by -
```bash
pip install PyQt5
```
```bash
pip install qt5-tools
pip install pyperclip

```
* Clone the repository
* Run the app by -
```bash
cd src
```
```bash
python main.py
```
